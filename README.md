#### Usage
ilcalc <opération> [...] ou ilcalc [...]
opérations:
  ilcalc {-a --aider}     ->  afficher cette page
  ilcalc {-c --config}    ->  modifier la configuration de l'application
  ilcalc {-r --résoudre}  ->  résoudre l'équation (commande optionelle)
